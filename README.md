# Glowing Button
A simple glowing button built with HTML and CSS. The button creates a glowing neon effect when hovered.

## Preview
A clean UI with a glowing hover animation.

## Features
- HTML & CSS only
- Smooth hover animation
- Neon glow effect
- Lightweight and beginner-friendly

## Technologies Used
- HTML5
- CSS3

## Project Structure
```.
├── index.html
└── index.css
```

## Getting Started
1. Clone the repository:
```bash
git clone https://github.com/your-username/Glowing-Button.git
```
2. Open the project folder.
3. Open `index.html` in your browser.
